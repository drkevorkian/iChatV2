1. Audit Logging & Compliance Exports (Top Priority ‚Äì Compliance Gold)
Enterprises (especially GDPR/HIPAA/SOC2 environments) require full audit trails.

Log every significant action: login/logout, message send/edit/delete, file upload/download, room join/leave, admin changes, moderation actions, failed logins, etc.
Timestamp, user ID/handle, IP, user-agent, action type, before/after values (for edits).

This is mandatory for SOC2 Type 2, HIPAA audit requests, and ISO 27001. Your current logging/rotation system is a great base ‚Äî extend it.
